核心项目

PROJECT · 001
生产企业销售-生产数据链路 AI 优化
PythonSQL数据自动化AI降本增效
在国有生产型企业主导"销售-生产数据链路的AI优化项目"。针对每月需要整合的 3类销售台账、5类生产日报表， 用 Python 结合 SQL 搭建了数据自动抓取、匹配与异常校验模块。 把原本需要跨 3 个部门、耗时 5 天的数据核对工作缩短到 2小时完成， 并实现了销量异动对生产计划的自动预警，帮助生产部门排产响应速度提升 70%， 成品库存周转率提高 22%。
